## Website

### Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Updating Projects

Projects are stored in `data/projects.json`. 

### Building for Production

```bash
npm run build
npm start
```
